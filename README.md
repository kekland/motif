# motif

An application to prototype UIs, draw vector graphics, and animate them all.

Currently very much in progress. The aim is to have a full Figma-like experience, but with additional features:

- No stupid paywalls for features that should be free
- Keyframed animation system
- Interactivity and scripting system
- WASM plugin support for custom features/integrations
- Full multiplayer support with an ability to self-host a server
- Eventually: exporting the design components to any frameworks (Flutter, React, HTML/CSS, etc.), with support for customizing the export templates.
